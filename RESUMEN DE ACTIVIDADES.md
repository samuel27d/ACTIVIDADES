# ACTIVIDADES
ACTIVIDADES DEL ESTUDIANTE SAMUEL COLMENAREZ
En este pequeño apartado se resumen todas las actividades realizadas hasta el momento:

Ejercicio 1:
En este primer ejercicio, el más simple de todos, llamado Sistema de Usuarios, se buscaba representar de manera básica y sencilla cómo funciona un sistema según el tipo de usuario que lo utilice, ya sea un administrador, un profesor o un estudiante, y las diferentes opciones que tiene cada uno de ellos.

Ejercicio 2:
En el Sistema del Zoológico, se muestra de forma básica y clara la presentación de cuatro animales del zoológico junto con los métodos de alimentación que los encargados utilizan con cada uno de ellos, finalizando con pequeños datos curiosos, como el método de escape de algunos y el método de caza de otros.
